# 나라 관보

대한민국 관보를 사람이 읽고 기계가 찾기 쉬운 형태로 탐색하는 정적 리더 프로토타입입니다.

## 실행

`index.html`을 브라우저에서 열면 됩니다. 로컬 서버가 필요하다면 원하는 정적 서버로 이 폴더를 서빙하세요.

## 포함 기능

- 제목·기관·본문 검색
- 기간·기관 필터 및 정렬
- 월별 기록 밀도 차트
- 목적별 진입점: 정책 변화 추적, 원문 근거 확인, 관심 기록 비교
- 문서 리더, 이전/다음 탐색, 링크 복사
- 브라우저에 유지되는 개인 저장함
- 다크 모드 및 키보드 단축키

현재 `app.js`의 `records`는 화면 검증을 위한 예시 데이터입니다. 실제 서비스에서는 관보 코퍼스의 `meta.json`, 날짜별 인덱스, Markdown 본문을 같은 형태로 연결하면 됩니다. 원문 링크는 문서마다 실제 전자관보 URL로 교체하세요.
