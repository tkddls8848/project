# 온라인 공개 전략 v0

## 전략 문장

`대한민국 제도 100`을 공개 웹서비스로 운영한다. 제도별 한 장 요약과 업무구조도를 검색 가능한 형태로 제공하고, 이용 흐름과 직접 피드백을 바탕으로 정보 구조와 검증 품질을 개선한다.

## 왜 웹서비스인가

- 한 장 구조도와 업무구조도는 실제 화면에서 탐색할 때 효용이 크다.
- 법령 개정에 맞춰 기준일과 검증 상태를 갱신할 수 있다.
- 제도명, 법령명, 절차명으로 검색 유입이 가능하다.
- 같은 데이터 계약으로 100개 제도를 비교할 수 있다.
- 공개 근거와 현장 검증 항목을 지속적으로 보완할 수 있다.

## 서비스 확장 단계

### 1단계: 대표 제도 공개

- 첫 10개 제도와 환경영향평가 기준 샘플
- 제도별 한 장 요약
- 상태 인식형 업무구조도
- 법령 근거와 기준일
- 제도 제작 요청 메일 초안

### 2단계: 카탈로그 확장

- 100개 제도 검색·필터·정렬
- 최대 3개 제도 비교
- 제도 유형과 분야별 탐색
- 데스크톱·모바일 공통 상세 화면

### 3단계: 검증 운영

- 공식 원문 연결과 조문 번호 확인
- 현장 검증 대장 공개
- 법령 최신성 주기 점검
- 오류 제보와 개선 의견 반영

## 공개 채널

- GitHub Pages 라이브 사이트
- 검색 엔진과 직접 링크
- 정책·행정 관련 온라인 커뮤니티
- 공무원, 보좌진, 연구자, 기자와 실무자 대상 직접 공유

## 핵심 화면

1. 100개 제도 카탈로그
2. 검색·필터·정렬
3. 제도 비교
4. 제도 상세와 업무구조도
5. 현장 검증 대장
6. 제도 제작 요청

## 품질 지표

- 검색 결과 0건 비율
- 제도 상세 진입과 업무구조도 이용량
- 비교 기능 사용량
- 많이 선택되는 분야와 유형
- 반복 요청되는 제도명
- 오류 제보와 검증 항목 해결 건수

## 포지셔닝

이 프로젝트는 제도 해설을 나열하는 블로그가 아니라, 법령·기관·절차·문서·예산을 같은 기준으로 구조화한 공개 제도 정보 서비스다.

## 운영 원칙

- 법령 기준일과 검증 범위를 명확히 표시한다.
- 공식 원문 확인과 현장 검증 필요를 구분한다.
- 신청자 정보와 비공개 내부 자료를 저장하지 않는다.
- 정적 배포와 반복 가능한 데이터 검증을 유지한다.
