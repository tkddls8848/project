- 지역 지정 필요: 9건

각 항목은 제도 JSON의 `verification.unresolved`에 `reasonCode`, `reason`, `nextStep`으로 기록한다.

## 방법

1. 캔버스 법적 근거 437개 고유 명칭을 국가법령정보센터 Open API로 검색했다.
2. 법률·시행령·시행규칙은 법령 ID와 MST, 행정규칙은 행정규칙 ID와 일련번호를 저장했다.
3. 캔버스와 절차 노드의 범위 조문과 가지조문을 파싱했다.
4. 법령 조문은 `target=law` JSON 전문, 행정규칙 조문은 `target=admrul` JSON 전문을 우선 조회하고 본문 미제공 시 Korean Law CLI로 보완 대조했다.
5. 상위법을 설명하기 위해 적힌 `법 제N조` 표기는 하위법령 조문으로 잘못 집계하지 않도록 제외했다.

## 한계

조문 검증은 현행 원문에 해당 조문 번호가 존재하는지를 확인한다. 인용한 설명이 조문의 법적 의미를 완전하게 반영하는지, 개별 사건에 적용되는지, 판례·유권해석과 일치하는지는 자동 검증 범위가 아니다. 지역 조례, 기관별 직제, 내부규정, 연도별 계획·편람은 적용 범위를 정한 후 별도 원문을 연결해야 한다.

기계 판독 보고서는 `verification-coverage.json`과 `article-verification-coverage.json`에 있다.
