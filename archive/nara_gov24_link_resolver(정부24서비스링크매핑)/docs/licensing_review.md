## 1. 수집 대상

- 정부24(gov.kr) 서비스 안내 페이지 URL
- 각 기관 홈페이지 및 민원 안내 링크
- CappBizCD 기반 서비스 식별자

## 2. 허용 범위

| 항목 | 판단 | 근거 |
|---|---|---|
| 공개 URL 저장 | 허용 | 공공기관 공개 정보 |
| 서비스 안내 링크 제공 | 허용 | 민원 안내 목적 |
| 자동 신청 자동화 | **금지** | 이용약관 위반 가능 |
| 본인인증 자동화 | **금지** | 전자서명법·개인정보보호법 |

## 3. 정부24 이용약관 확인 항목 (미완료)

- [ ] 정부24 이용약관 §크롤링/자동수집 조항 확인

- [ ] CappBizCD 공개 여부 확인

## 4. 자동 수집 가능 범위 (잠정)

현재 MVP는 **수동 seed 링크**만 수집한다.  
자동 크롤러 추가 전 반드시 이 문서를 업데이트하고 법적 범위를 재확인한다.
