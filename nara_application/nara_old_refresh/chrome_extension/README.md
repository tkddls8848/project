# 공공데이터포털 자동 승인 크롬 익스텐션

공공데이터포털(data.go.kr)의 API 활용 신청을 자동으로 연장 승인하는 크롬 익스텐션입니다.

## 주요 기능

- **자동 로그인 확인**: 공공데이터포털 로그인 상태 자동 확인
- **전체 페이지 순회**: 목록의 모든 페이지를 자동으로 순회
- **자동 연장 신청**: 각 API 활용 신청에 대해 자동으로 연장 신청 처리
- **실시간 진행 상황**: 처리 중인 항목과 진행률을 실시간으로 표시
- **상세 로그**: 모든 처리 과정을 로그로 기록
- **설정 가능**: 자동 확인, 대기 시간, 오류 처리 등 다양한 옵션 설정

## 설치 방법

### 1. 익스텐션 파일 준비

이 폴더의 모든 파일이 준비되어 있습니다.

```
chrome_extension/
├── manifest.json
├── background.js
├── content.js
├── popup.html
├── popup.js
├── styles.css
└── README.md
```

### 2. 크롬에 익스텐션 로드

1. 크롬 브라우저를 엽니다
2. 주소창에 `chrome://extensions/` 입력
3. 우측 상단의 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. `chrome_extension` 폴더 선택

### 3. 익스텐션 확인

- 크롬 확장 프로그램 목록에 "공공데이터포털 자동 승인"이 표시됩니다
- 주소창 오른쪽에 익스텐션 아이콘이 표시됩니다 (기본 아이콘)

## 사용 방법

### 1. 공공데이터포털 로그인

1. [공공데이터포털](https://www.data.go.kr)에 접속
2. 계정으로 로그인

### 2. 자동 승인 실행

1. 로그인 후 익스텐션 아이콘 클릭
2. 팝업 창에서 설정 확인
   - **확인 대화상자 자동 승인**: 연장 신청 시 확인 대화상자를 자동으로 승인
   - **항목 간 대기 시간**: 각 항목 처리 후 1초 대기 (서버 부하 방지)
   - **오류 발생 시 중지**: 오류 발생 시 자동으로 프로세스 중지
3. "자동 승인 시작" 버튼 클릭
4. 자동으로 목록 페이지로 이동하여 처리 시작

### 3. 진행 상황 확인

팝업 창에서 다음 정보를 실시간으로 확인할 수 있습니다:
- **상태**: 현재 프로세스 상태
- **현재 페이지**: 처리 중인 페이지 번호
- **처리된 항목**: 총 처리된 항목 수
- **성공/실패**: 성공 및 실패 항목 수
- **진행률**: 현재 페이지의 처리 진행률
- **처리 로그**: 상세 처리 내역

### 4. 중지 및 초기화

- **중지**: 진행 중인 프로세스를 즉시 중지
- **초기화**: 통계 및 진행률 초기화

## 동작 원리

1. **로그인 확인**: 페이지에서 로그아웃/마이페이지 요소 확인
2. **목록 수집**: `selectAcountList.do` 페이지에서 모든 API 활용 신청 항목 수집
3. **상세 페이지 이동**: 각 항목의 링크를 클릭하여 상세 페이지로 이동
4. **연장 신청 처리**: `fn_reqst('extend', '연장')` 함수 실행
5. **확인 대화상자 처리**: confirm/alert 대화상자 자동 처리
6. **결과 확인**: 페이지 이동 및 메시지로 성공 여부 확인
7. **다음 항목**: 목록 페이지로 돌아가 다음 항목 처리
8. **페이지 순회**: 모든 페이지를 순회하며 위 과정 반복

## 기술 스택

- **Manifest V3**: 최신 크롬 익스텐션 표준
- **Content Script**: 페이지 DOM 조작 및 자동화
- **Background Service Worker**: 백그라운드 이벤트 처리
- **Chrome Storage API**: 설정 및 통계 저장
- **Chrome Messaging API**: 컴포넌트 간 통신

## 주의사항

1. **로그인 필수**: 익스텐션 사용 전 공공데이터포털에 로그인해야 합니다
2. **페이지 유지**: 자동화 진행 중 페이지를 닫거나 이동하지 마세요
3. **네트워크 안정성**: 안정적인 네트워크 환경에서 사용하세요
4. **서버 부하**: 대량의 항목이 있는 경우 시간이 오래 걸릴 수 있습니다
5. **오류 처리**: 오류 발생 시 로그를 확인하고 필요시 수동으로 처리하세요

## 문제 해결

### 익스텐션이 작동하지 않는 경우

1. **페이지 새로고침**: 공공데이터포털 페이지를 새로고침하세요
2. **익스텐션 재로드**: `chrome://extensions/`에서 익스텐션을 재로드하세요
3. **로그인 확인**: 로그인 상태를 확인하세요
4. **콘솔 확인**: F12를 눌러 개발자 도구의 콘솔 탭에서 오류 메시지 확인

### 로그에 오류가 표시되는 경우

- **로그인이 필요합니다**: 공공데이터포털에 로그인하세요
- **fn_reqst 함수를 찾을 수 없습니다**: 페이지 구조가 변경되었을 수 있습니다
- **상세 페이지 이동 실패**: 네트워크 문제이거나 페이지 구조가 변경되었을 수 있습니다

### 성능 최적화

- **대기 시간 조정**: 설정에서 "항목 간 대기 시간"을 비활성화하여 처리 속도 향상 (권장하지 않음)
- **오류 발생 시 중지**: 오류가 많이 발생하는 경우 활성화하여 불필요한 처리 방지
